## No Data Collection

This project does **not** collect, transmit, or store any personal data or usage telemetry. No analytics, tracking, or logging are performed by default.

## Security Responsibilities

* **Users:** Use this project at your own risk. You are responsible for verifying its suitability, applying updates, and operating it securely in your environment.
* **Maintainers:** We will make reasonable efforts to address confirmed security issues in a timely manner.

## Reporting Vulnerabilities

If you discover a security issue, please **open a new GitHub Issue** in this repository and label it as **“security”**.
Do **not** include detailed exploits in the public issue. Instead, describe the problem in general terms and a maintainer will follow up with next steps.
